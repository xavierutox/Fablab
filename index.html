    Introducción

    La siguiente es la documentación de la API de Reportes de Maquina, se han creado 5 endpoints para la entrega que serán enumeradas y explicadas a continuación. Para la entrega y documentación de la API se eligió la herramienta Open Source de Slate

    Para el uso y consumo de la API se utiliza Axios, la cual se puede instalar con
    npm install axios

    API Obtener Reportes

    Reportes /

    Utilizar Axios para poder consumir la información

    
const axios = require('axios');

axios.get("http://18.116.81.54:4000/Reportes", {
        params: {
            "email": "camila.pinochet@sansano.usm.cl",
            "filtro": "u",
        }}).then((res) => {
            let data = res.data;
            console.log(data);
        })
        .catch((err) => {
            console.log(err);
        });

    
import requests

# reporte de usuarios

endpoint= ''

      "account": "camila.pinochet@sansano.usm.cl",
      "print_errors": 0,
      "prints": 1,
      "real_print_time_h": 0.66,
      "material_used_kg": 0.005
    }
  ],
  "billing": [],
  "impresiones": []
}

    Devuelve el reporte de un usuario; su consumo, impresiones y detalles de su uso.

    HTTP Request

    GET http://18.116.81.54:4000/Reportes/

    Query Parameters

    Parametros Descripción
    email Si se añade un email se puede filtrar por email.
    filtro Si se añade u, b, i, permite filtrar por Usos, Consumos (billing) e Impresiones, respectivamente.
    Se pueden añadir varios filtros a la vez separados por una ",", ejemplo: "...?filtro=u,i"

            console.log(err);
        });

    
import requests

endpoint= 'Usuarios'
url = f'http://18.116.81.54:4000/Reportes/{endpoint}'
response = requests.get(url, params={
    'email': 'josue.venegas@sansano.usm.cl',

    Esta llamada entrega una respuesta del siguiente formato:

    [
  {

    workgroup_name: 'FABLAB',

    Devuelve los datos sobre los usuarios y el grupo al que pertenece, además de si es o no un administrador.

    HTTP Request

    GET http://18.116.81.54:4000/Reportes/Usuarios

    Query Parameters

    Parametros Descripción
    email Si se añade un email se puede filtrar por email.
    grupo Si se añade o existe un nombre de grupo, permite filtrar por el nombre del mismo.
    Omitir un parametro omite el filtro.

    HTTP Request

    GET http://18.116.81.54:4000/Reportes/Cancelaciones

    Errors

    La API de Reportes presenta los siguientes manejos de errores:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    404 Not Found -- The specified endpoint could not be found.
